# Extract item parameters as data frame

Extract item parameters as data frame

## Usage

``` r
# S3 method for class 'item_params'
as.data.frame(x, row.names = NULL, optional = FALSE, ...)
```

## Arguments

- x:

  An `item_params` object.

- row.names:

  NULL or a character vector giving the row names.

- optional:

  Logical. Compatibility argument for
  [`as.data.frame()`](https://rdrr.io/r/base/as.data.frame.html);
  currently ignored.

- ...:

  Additional arguments (ignored).

## Value

A data frame of item parameters.
