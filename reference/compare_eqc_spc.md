# Deprecated Alias for EQC/SAC Comparison

`compare_eqc_spc()` is a deprecated backward-compatible alias for
[`compare_eqc_sac()`](https://joonho112.github.io/IRTsimrel/reference/compare_eqc_sac.md).
It is retained for scripts written against the v0.1.x naming convention
and will issue a deprecation warning when called.

## Usage

``` r
compare_eqc_spc(...)
```

## Arguments

- ...:

  Arguments passed to
  [`compare_eqc_sac()`](https://joonho112.github.io/IRTsimrel/reference/compare_eqc_sac.md).

## Value

The result of
[`compare_eqc_sac`](https://joonho112.github.io/IRTsimrel/reference/compare_eqc_sac.md).
