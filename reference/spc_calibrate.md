# Deprecated Alias for SAC Calibration

`spc_calibrate()` is a deprecated backward-compatible alias for
[`sac_calibrate()`](https://joonho112.github.io/IRTsimrel/reference/sac_calibrate.md).
It is retained for scripts written against the v0.1.x naming convention
and will issue a deprecation warning when called.

## Usage

``` r
spc_calibrate(...)
```

## Arguments

- ...:

  Arguments passed to
  [`sac_calibrate()`](https://joonho112.github.io/IRTsimrel/reference/sac_calibrate.md).

## Value

The result of
[`sac_calibrate`](https://joonho112.github.io/IRTsimrel/reference/sac_calibrate.md).
