# Print Method for summary.item_params Objects

Print Method for summary.item_params Objects

## Usage

``` r
# S3 method for class 'summary.item_params'
print(x, digits = 4, ...)
```

## Arguments

- x:

  A `summary.item_params` object from
  [`summary.item_params()`](https://joonho112.github.io/IRTsimrel/reference/sim_item_params.md).

- digits:

  Integer. Number of decimal places for printing.

- ...:

  Additional arguments (ignored).

## Value

The input object, invisibly.
